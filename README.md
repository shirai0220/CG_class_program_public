## 作品概要
この作品は、Three.js(WebGL)を用いた「立体シャッフルカップゲーム」です。

大学の講義で学んだ3Dグラフィックスの基礎をベースに、blenderにて作った「ゴルフボール」と「正二十面体」を合成して、ゴルフボールにとんがりが複数ついているようなオブジェクトAが、青い立方体の中に入り、その立方体が動き回ることで、最終的にオブジェクトAがどこにあるのかを見つけるゲームを作成しました。
